## Vision

Face auth for Linux that's as easy and reliable as Windows Hello.

## v0.2 - Ease of Use (current focus)

- [ ] IR camera auto-detection (scan V4L2 devices)
- [ ] `faceauth setup` wizard (one-command install)
- [ ] Audit logging (structured JSON-lines auth trail)
- [ ] README rewrite (comparison vs Howdy, demo, simplified quick start)
- [ ] PyPI package (`pip install faceauth`)

## v0.3 - Extensibility

- [ ] Pluggable recognition backend (open-licensed model option)
- [ ] Hardware compatibility database
- [ ] Plugin API for custom anti-spoof models

## v0.4 - Enterprise

- [ ] Encryption at rest for embeddings
- [ ] Multi-user enrollment management
- [ ] Centralized configuration deployment
